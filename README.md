# 📋 Sobre o Projeto

Este projeto consiste em uma aplicação web desenvolvida utilizando **HTML, CSS e JavaScript puro**, com o objetivo de simular um sistema de cadastro e gerenciamento de contas de jogos online.

O sistema possui uma tela de login e, após a autenticação, permite realizar operações completas de CRUD (Create, Read, Update e Delete) sobre uma lista de contas de jogos.

A solução escolhida foi a gerada pelo **ChatGPT**.
